zlib for XCode
==========

This repository depends https://github.com/madler/zlib

usage
-----

1. clone and move repository

    ```
    https://github.com/leico/zlib-xcode
    cd zlib-xcode
    ```
1. move and clone zlib submodule

    ```
    git submodule update --init --recursive
    ```

1. build zlib
